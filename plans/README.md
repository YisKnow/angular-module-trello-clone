## Quick wins (no plan needed)

These are one-line fixes that any contributor can apply directly:

- **`.nvmrc`** → change `18` to `22` (or `20`). Angular 22 needs Node ≥18.19.1
  or ≥20.11.0; `nvm use 18` resolves to 18.0.0 which is too old.
- **`eslint.config.js`** → remove `'@angular-eslint/prefer-standalone': 'off'`.
  The app is 100% standalone; this rule should be active.
- **`package.json` no `test:watch` parity** — `test:watch: "vitest"` exists, but
  `test` uses `vitest run`. Consider adding `"test:coverage": "vitest run --coverage"`.

  and error feedback; schedule after Plans 001 and 002.

  deterministic network scenarios.
- **Unify Signal Forms** across the app: medium effort, medium value for
  consistency. `BoardComponent` still uses `ReactiveFormsModule`. Defer to a
  future standards plan.
- **Card-dialog and placeholder-navigation features**: product work, not part of
